# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/35e5f262-a2c5-4fb1-b512-2082dde5abaa

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

গুগল শিট ডাটাবেজ কানেক্ট করা এবং এআই সিস্টেম নিয়ে বিস্তারিত গাইড নিচে দেওয়া হলো:

আপনি গুগল অ্যাপস স্ক্রিপ্ট (Google Apps Script) ব্যবহার করে খুব সহজেই আপনার কুইজ প্ল্যাটফর্মের ব্যাকএন্ড ডাটাবেজটি গুগল শিটের সাথে সরাসরি লিংক করতে পারবেন।
ধাপ ১: গুগল শিট (Google Sheet) ও ট্যাব প্রস্তুত করা
প্রথমে আপনার ব্রাউজারে একটি নতুন Google Sheet তৈরি করুন।

ফলাফল
সার্টিফিকেটসমূহ
নোটিফিকেশনসমূহ
ধাপ ২: অ্যাপস স্ক্রিপ্ট (Apps Script) কোড ও Spreadsheet ID সংযুক্তকরণ
আপনার ওপেন করা গুগল শিটের ওপরের মেনু বার থেকে Extensions 
 Apps Script-এ প্রবেশ করুন।
কোড এডিটরে পূর্বে থাকা সমস্ত লেখা মুছে ফেলুন।
সেটিংস প্যানেলের নির্দেশগায় থাকা Code.gs source code here লিংকে ক্লিক করে আমাদের কাস্টম অ্যাপস স্ক্রিপ্ট কোডটি কপি করে সেখানে পেস্ট করুন।
কোডটির একেবারে শুরুতে থাকা নিচের লাইনটি লক্ষ্য করুন:
code
JavaScript

ধাপ ৩: অ্যাপস স্ক্রিপ্ট ওয়েব অ্যাপ হিসেবে প্রকাশ (Deployment)
স্ক্রিপ্ট এডিটরের ওপরের সেভ আইকনে ক্লিক করে কোডটি সেভ করুন।
ডানদিকের ওপরে Deploy 
 New Deployment বাটনে ক্লিক করুন।
গিয়ার আইকনে ক্লিক করে ড্রপডাউন থেকে Web App নির্বাচন করুন।
কনফিগারেশনে নিচের অপশন দুটি একদম হুবহু সিলেক্ট করবেন:
Execute as: Me (আপনার ইমেইল আইডি)
Who has access: Anyone (অবশ্যই "Anyone" সিলেক্ট করতে হবে যেন ওয়েবসাইট থেকে রিকোয়েস্ট সাবমিট হতে পারে)
Deploy বাটনে ক্লিক করুন এবং গুগল চাওয়া অনুমতিসমূহ (Permissions) অনুমোদিত (Authorize Access) করুন।
ডিপ্লয়মেন্ট সফল হওয়ার পর স্ক্রিনে একটি দীর্ঘ Web App URL দেখতে পাবেন। সেটি সম্পূর্ণ কপি করে নিন।
ধাপ ৪: স্মার্ট কুইজ প্রো প্ল্যাটফর্মে লিঙ্ক করা
আপনার কুইজ প্ল্যাটফর্মের ডানদিকের ওপরের অ্যাডমিন ড্যাশবোর্ড (Admin Dashboard) বাটনে ক্লিক করুন (সিকিউরিটি কী দিয়ে লগইন করুন)।
বা-দিকের নেভিগেশন মেনু থেকে একদম নিচে অবস্থিত সেটিংস প্যানেল (Settings Panel)-এ প্রবেশ করুন।
গুগল শিট ডাটাবেজ লিঙ্কার সেকশনে থাকা ইনপুট বক্সে আপনার কপি করা Web App URL-টি পেস্ট করুন।
